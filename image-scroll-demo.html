图片横向流动效果

类似 midjourneysref.com 的图片展示效果

技术实现原理

这个图片流动效果主要使用了以下技术：

1. CSS Transform 和 Animation

@keyframes scrollLeft { 0% { transform: translateX(0); } 100% { transform: translateX(-50%); } } .image-flow-wrapper { animation: scrollLeft 60s linear infinite; }

2. 无缝循环的关键

为了实现无缝循环，我们需要：

  • 复制一份完整的图片组
  • 动画移动距离设置为容器宽度的50%
  • 当动画结束时，视觉上回到起始位置

3. 性能优化

重要提示： 使用 transform: translateX() 而不是改变 left 属性，因为 transform 会触发 GPU 加速，性能更好。

2. 关键CSS样式

.image-flow-container { position: relative; height: 460px; overflow: hidden; } .image-flow-wrapper { display: flex; position: absolute; animation: scrollLeft 60s linear infinite; } .image-column { width: 180px; flex-shrink: 0; display: flex; flex-direction: column; }

3. JavaScript控制（可选）

// 暂停动画 function pauseAnimation() { document.getElementById('imageWrapper').style.animationPlayState = 'paused'; } // 继续动画 function resumeAnimation() { document.getElementById('imageWrapper').style.animationPlayState = 'running'; } // 改变速度 function changeSpeed(speed) { const wrapper = document.getElementById('imageWrapper'); const speeds = { 'slow': '120s', 'normal': '60s', 'fast': '30s' }; wrapper.style.animationDuration = speeds[speed]; }
