## 技术细节

### 动画原理
使用CSS `transform: translateX()` 和 `@keyframes` 实现无缝循环滚动：
```css
@keyframes scrollLeft {
  0% { transform: translateX(0); }
  100% { transform: translateX(-50%); }
}
```

### 速度控制
通过调整 `animation-duration` 属性控制流动速度：
- 数值越大，流动越慢
- 数值越小，流动越快

### 间隔调整
通过CSS的 `margin-right` 属性控制图片列之间的间距：
- React组件：通过 `gap` prop 控制
- HTML版本：直接修改CSS类
