# 阶段12：首页图片展示区域视觉突出优化

**日期**：2026-03-05
**分支**：main
**状态**：✅ 完成

---

## 一、背景与问题分析

### 1.1 现象

首页结构分为三层：
1. **Hero 区**：全幅图片背景 + 居中文字 CTA，视觉强烈
2. **Explore Collections**：3 张入口卡片，白色背景
3. **Latest Content**：搜索筛选 + 瀑布流图片网格

问题：第 3 层（图片瀑布流）与第 2 层（入口卡片）同处白色背景，两个区域缺乏视觉区分，导致：
- 用户滚动时难以感知进入了"展览区"
- 图片网格缺少"舞台感"，AI 作品的视觉冲击力未得到充分展示
- 搜索/筛选工具栏与图片网格之间没有明确分隔

### 1.2 设计目标

- 让 Latest Content 区域在视觉上成为独立的"画廊区域"
- 不改变整体亮色主题风格
- 改动克制，只用最少的 CSS 实现最大的层次感提升

---

## 二、方案选型

| 方案 | 描述 | 选择理由 |
|------|------|----------|
| A — 深色背景 | 给图片区换成深色背景 | ❌ 改变整体主题风格，用户明确表示是亮色主题 |
| B — 灰色背景 + 分隔线 | 用 `--bg-secondary` 轻微区分 + border | ✅ 克制、有效、主题一致 |
| C — 卡片阴影强化 | 只调整卡片 shadow | ❌ 影响范围小，无法解决区域感知问题 |

**最终采用方案 B**，配合 accent 竖线标题和筛选行底部分隔线，三重措施叠加。

---

## 三、具体改动

### 3.1 Latest Content 区域底色 + 顶部边框

```css
/* Before */
.home-content-section {
  max-width: none;
  margin-left: 0;
  margin-right: 0;
  margin-bottom: 0;
  padding: 0 0.75rem;
}

/* After */
.home-content-section {
  max-width: none;
  margin-left: 0;
  margin-right: 0;
  margin-bottom: 0;
  padding: 2.5rem 0.75rem 0;
  background-color: var(--bg-secondary);   /* 亮色: #f8fafc / 暗色: #111827 */
  border-top: 1px solid var(--border-color); /* 与 Collections 区的明确分界线 */
}
```

**效果**：Collections 区（白色）→ 分割线 → Latest Content 区（浅灰），用户滚动时能感知到进入了不同区域。

### 3.2 Section Header 标题前 accent 竖线

只对 `.home-content-section` 内的标题加竖线，Explore Collections 不受影响：

```css
/* Latest Content 区域标题前加彩色竖线 accent */
.home-content-section .home-section-header h2 {
  display: flex;
  align-items: center;
  gap: 0.6rem;
}

.home-content-section .home-section-header h2::before {
  content: '';
  display: inline-block;
  width: 4px;
  height: 1.4em;
  border-radius: 2px;
  background: linear-gradient(180deg, #6366f1, #a855f7);  /* 品牌色渐变 */
  flex-shrink: 0;
}
```

**效果**：标题左侧出现紫色渐变竖线 accent，与品牌色一致，让"Latest Content"标题区有明显的视觉锚点。

### 3.3 筛选行底部分隔线

```css
/* Before */
.gallery-filters-row {
  display: flex;
  align-items: center;
  gap: 0.5rem;
  flex-wrap: wrap;
}

/* After */
.gallery-filters-row {
  display: flex;
  align-items: center;
  gap: 0.5rem;
  flex-wrap: wrap;
  padding-bottom: 0.75rem;
  border-bottom: 1px solid var(--border-color);
  margin-bottom: 0.25rem;
}
```

**效果**：搜索/标签筛选工具栏与图片瀑布流之间有清晰分隔，视觉层级：标题 → 搜索 → 筛选 → [分割线] → 图片。

### 3.4 移除 home-dark-area 底部 padding

```css
/* Before */
.home-dark-area {
  padding: 3rem 0 4rem;
}

/* After */
.home-dark-area {
  padding: 3rem 0 0;
}
```

原来底部 `4rem` 的 padding 在图片结束后产生大块空白，去掉后图片网格直接延伸至页脚，符合画廊满版展示的设计惯例。

### 3.5 响应式适配

```css
@media (max-width: 768px) {
  .home-dark-area {
    padding: 2rem 0 0;    /* 同步去掉底部 padding */
  }

  .home-content-section {
    padding-top: 1.75rem; /* 移动端顶部内距略小 */
  }
}
```

## 四、视觉效果对比

| 区域 | 改动前 | 改动后 |
|------|--------|--------|
| Collections → Content 过渡 | 无边界，连续白色背景 | 1px 分割线 + 背景色切换，明确区域边界 |
| Latest Content 标题 | 普通文字标题 | 左侧紫色竖线 accent + 标题文字 |
| 筛选栏与图片 | 无分隔 | 底部 1px border 明确工具区与展示区 |
| 图片区底部 | 大块白色空白 | 直接与页脚相接，满版展示 |

## 五、主题兼容性

所有新增样式均使用 CSS 变量：

- `var(--bg-secondary)` — 亮色 `#f8fafc`，暗色 `#111827`
- `var(--border-color)` — 跟随主题自动切换
- 竖线渐变色（`#6366f1` → `#a855f7`）在亮色/暗色下均可见
